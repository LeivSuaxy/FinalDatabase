Bienvenidos al proyecto final de Base de Datos

Esta pagina web, ha sido creada con el proposito fundamental de darle solucion a la problematica planteada en el proyecto final de la asignatura Base de Datos I.

El problema a desarrollar fue el numero 2, El objetivo de este proyecto es desarrollar un sistema de reservas para un hotel. La aplicación permitirá a los clientes hacer reservas en línea, a los empleados administrar las reservas y a los gerentes supervisar el funcionamiento del hotel. Los requerimientos para este proyecto son los siguientes:

  • Los clientes deben ser capaces de hacer reservas en línea para una o más habitaciones y de cancelar sus reservas.
  • Los empleados deben ser capaces de ver todas las reservas, cancelar o modificar reservas, y verificar la disponibilidad de habitaciones.
  • Los gerentes deben ser capaces de ver estadísticas y reportes sobre las reservas, las habitaciones y los ingresos del hotel.
  • La aplicación debe permitir a los clientes crear una cuenta y autenticarse para acceder a la funcionalidad de la aplicación.
  • Cada reserva debe contener información sobre la fecha de entrada, la fecha de salida, el número de huéspedes y las habitaciones reservadas.
