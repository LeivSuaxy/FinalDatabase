Bienvenidos al proyecto final de Sistemas de Bases de Datos

Esta pagina web, ha sido creada con el proposito fundamental de darle solucion a la problematica planteada en el proyecto final de la asignatura Base de Datos I.

El problema a desarrollar fue el numero 2. El objetivo de este proyecto es desarrollar un sistema de gestión de inventario para una tienda minorista. La aplicación permitirá a los empleados gestionar el inventario de productos, procesar las órdenes de compra, y generar reportes de ventas y de inventario. Los requerimientos para este proyecto son los siguientes:

  • • Los empleados deben ser capaces de agregar, actualizar y eliminar productos en el inventario.
  • • Los empleados deben ser capaces de procesar órdenes de compra y actualizar el inventario en consecuencia.
  • • Los empleados deben ser capaces de generar reportes de ventas y de inventario.
  • • La aplicación debe permitir a los empleados crear una cuenta y autenticarse para acceder a la funcionalidad de la aplicación.
  • • Cada producto en el inventario debe contener información sobre el nombre del producto, el precio, la cantidad en stock y la descripción.
