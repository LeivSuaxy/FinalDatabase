Welcome to the final project of Database Systems

This website has been created with the fundamental purpose of providing a solution to the problem raised in the final project of the subject Database I.

The problem to be developed was number 2. The objective of this project is to develop a system inventory management for a retail store. The application will allow employees to manage inventory of products, process purchase orders, and generate sales and inventory reports. The requirements for This project are the following:

  • • Employees must be able to add, update and delete products on the inventory.
  • • Employees must be able to process purchase orders and update inventory accordingly.
  • • Employees must be able to generate sales and inventory reports.
  • • The app must allow employees to create an account and authenticate to access the functionality of the application.
  • • Each product in the inventory must contain information about the name of the product, the price, the quantity in stock and the description.
